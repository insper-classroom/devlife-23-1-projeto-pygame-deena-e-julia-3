Goldenfly

Sobre o jogo:

O nosso jogo é uma adaptação do quadribol de Harry Potter ao Flappy Bird. É um jogo cujo qual possui uma bola de ouro que deve ultrapassar o arco para pontuar e, caso contrário, finaliza o jogo com o seu recorde de pontuação armazenado. Finalmente, o objetivo do jogo é o usuário conseguir bater o recorde passado continuamente.
